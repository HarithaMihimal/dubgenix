Project Overview

The Challenge

In Sri Lanka, where Sinhala is the primary language, there's a significant gap in multilingual accessibility. While English content is abundant, Sinhala speakers often rely on subtitles, which can be distracting and limit accessibility for those with reading difficulties.

Current solutions focus primarily on subtitles, but real-time dubbing remains largely unexplored, especially for low-resource languages like Sinhala.

Our Goal

Develop a real-time system that can dub English videos to Sinhala, providing natural and intelligible speech output for better accessibility.

Multilingual Accessibility

Bridging the language gap in Sri Lanka through innovative AI-powered dubbing technology

System Architecture

Three-phase development approach for comprehensive dubbing system

Phase 1 – Core Dubbing Pipeline

  • Audio extraction and preprocessing
  • Noise removal and enhancement
  • ASR with ABC_ASR
  • MT with NLLB-1.3B
  • TTS with XTTS
  • Merging dubbed audio

Foundation

Building the core pipeline for audio processing, translation, and synthesis

Phase 2 – Real-Time Dubbing

  • Audio chunking and streaming
  • WebSocket-based pipeline
  • Latency optimization
  • Real-time processing

Performance

Optimizing for real-time processing with minimal latency

Phase 3 – Advanced AI Features

  • Speaker Diarization
  • Voice Cloning
  • Emotion Detection
  • Advanced AI integration

Intelligence

Advanced AI features for enhanced user experience
